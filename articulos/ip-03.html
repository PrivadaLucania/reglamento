IP-03 — Periodicidad de pintura de fachadas

Página 6 del PDF oficial

Texto original (literal)

4. Se deberá realizar el pintado de la fachada, por lo menos cada 3 años (iniciando el conteo en
2023), dado que el administrador tiene que autorizar los códigos de color.

Interpretación jurídica correcta

Se impone una obligación de mantenimiento periódico (repintado) mínimo cada 3 años, contabilizando desde 2023. Además, el código de color requiere autorización del Administrador. Es una medida para conservar uniformidad y evitar deterioro visible que afecte la imagen y plusvalía.

  • Condición: antes de pintar, se valida el código de color con administración.
  • Enfoque: mantenimiento preventivo, no “cambio de imagen”.

Ejemplos y escenarios

  • ✅ Correcto: repintar en 2026 si el conteo inició en 2023, usando el color autorizado.
  • ✅ Correcto: solicitar a administración el código antes de comprar pintura.
  • ⚠️ Riesgo: “pintar por mi cuenta” sin validación puede terminar en un tono distinto.
  • ❌ Incumplimiento: dejar deterioro visible prolongado, o usar color no autorizado.

Sanción aplicable

Estado: se vincula a “afectar fachadas” (ALTO) si hay alteración/impacto visible

El Tabulador no trae un renglón específico “no pintar cada 3 años”, pero si el incumplimiento deriva en afectar el aspecto de las fachadas (deterioro visible o color fuera de catálogo), encuadra como ALTO.

1ª sanción$1,500.00
2ª sanción$3,000.00

Mejor práctica: si hay desgaste, coordinar el repintado con administración para evitar observaciones y rehacer trabajos.
